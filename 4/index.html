Ejercicio 4: Validación XML mediante DTD y XSD

Construir:

  1. El vocabulario para el documento XML que utiliza esta empresa para gestionar los pedidos utilizando un DTD externo.

    2. Se puede abrir el documento DTD mediante este enlace.

  2. Ese mismo vocabulario utilizando el lenguaje XMLSchema debidamente documentado con fecha de creación, nombre del autor y utilidad del esquema.

    3. Se puede abrir el documento XSD mediante este enlace.

  3. Realizar un fichero XML que se corresponda con una instancia del vocabulario diseñado y asociarle al DTD.

    4. Se puede abrir el documento XML verificado con DTD mediante este enlace.

  4. Modificar ese fichero XML para asociarle el esquema XSD diseñado.

    5. Se puede abrir el documento XML verificado con XSD mediante este enlace.


Incluimos también la comprobación de formato y validación usando XML Copy Editor
